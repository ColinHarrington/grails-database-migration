dbm-list-locks

Purpose

Lists who currently has locks on the database changelog to STDOUT or a file.

Description

Usage: 

grails [environment] dbm-list-locks [filename] --defaultSchema=[defaultSchema]

Required arguments: none .

Optional arguments:

  • filename - The path to the output file to write to. If not specified output is written to the console
  • defaultSchema - The default schema name to use

Note that the defaultSchema parameter name and value must be quoted if executed in Windows, e.g. 
grails dbm-list-locks "--defaultSchema=[defaultSchema]"
